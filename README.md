## 📂 Repository Structure

* **`heart.csv`**: Dataset for the cardiovascular health/heart disease classification case study.
* **`insurance.csv`**: Dataset for the insurance premium cost prediction regression case study.
* **`Untitled.ipynb` / `Untitled1.ipynb`**: Jupyter Notebooks containing the source code, granular data analysis, and visualizations for the respective case studies.

## 🛠️ Tech Stack & Libraries

The following core Python libraries were utilized to manipulate the data, handle statistical computations, and generate visualizations:

* **Data Manipulation:** `pandas`, `numpy`
* **Data Visualization:** `seaborn`, `matplotlib`
* **Environment:** Jupyter Notebook

---

## ⚙️ Machine Learning Pipeline

To ensure high-quality data before feeding it into machine learning algorithms, both projects strictly adhere to the following pipeline:

1. **Exploratory Data Analysis (EDA):** Visualizing and summarizing the datasets statistically to uncover underlying patterns, correlations, and anomalies.
2. **Data Cleaning:** Handling missing values, isolating duplicate records, and addressing outliers.
3. **Data Preprocessing:** Transforming raw data into an algorithmic-friendly format (e.g., encoding categorical variables, handling skewness, and scaling).
4. **Feature Engineering:** Extracting, combining, or transforming variables to create optimal features for final model training.
